# exrna-mapper
Map ExRNA to proteins and interactions

A component for the Genboree workbench.

## To install

```
git clone git@github.com:ariutta/exrna-mapper.git
cd exrna-mapper
pip install -e .
pip install https://github.com/ariutta/networkxxgmml/zipball/master
```
## To run

```
cd exrnamapper
python __init__.py
```
